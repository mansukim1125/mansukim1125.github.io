# 카드

N장의 카드가 있다. 카드는 1부터 N까지의 숫자가 순서대로 적혀 있으며, 1이 가장 위에, N이 가장 아래에 위치해 있다. 이 상태에서 다음의 과정을 카드가 1장 남을 때까지 반복한다.

먼저, N장의 카드 중에서 제일 위에 있는 카드를 바닥에 버리고 맨 위에 있는 카드를 맨 아래에 있는 카드 아래로 옮긴다.

# 설명

1장 남은 카드에 적힌 숫자를 반환한다. 다음은 입·출력 예시이다.

# 입·출력 예시

| 입력 | 출력 |
| ---- | ---- |
| 4    | 4    |
| 6    | 4    |
